# PetAI Pro 官方网站

PetAI Pro 的官方网站，用于 App Store 上架所需的隐私政策、服务条款和技术支持页面。

## 网站地址

部署后可通过以下地址访问：
- 首页: https://athenalion007.github.io/petai-website/
- 隐私政策: https://athenalion007.github.io/petai-website/privacy.html
- 服务条款: https://athenalion007.github.io/petai-website/terms.html
- 技术支持: https://athenalion007.github.io/petai-website/support.html

## 部署说明

1. 在 GitHub 创建新仓库，命名为 `petai-website`
2. 将本目录下所有文件推送到该仓库
3. 在仓库设置中启用 GitHub Pages
4. 选择 main 分支作为部署源

## 文件说明

- `index.html` - 营销主页
- `privacy.html` - 隐私政策页面
- `terms.html` - 服务条款页面
- `support.html` - 技术支持页面
- `_config.yml` - GitHub Pages 配置文件
